Download for VSCode (.zip) Download for IntelliJ / PhpStorm (.zip)

VSCode Installation

  1. Extract the zip to ~/.vscode/extensions/cool-vscode/
  2. Restart VSCode
  3. Press Ctrl+Shift+P, type Color Theme, and select cool

IntelliJ / PhpStorm Installation

  1. Download the .zip and extract cool.icls
  2. Copy to your IDE config: ~/.config/JetBrains/<IDE>/colors/
  3. Restart and select cool under Settings → Editor → Color Scheme
